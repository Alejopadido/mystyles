Code Description
* {
margin: 0;
padding: 0;
box-sizing: border-box;
} 		This CSS rule, removes the default styles of the website, for example the margin and padding, this with the objetive that you won't have problems when you're positioning your elements. And also the box-sizing: border-box; makes that the total width and height of each element, is the product of the sum between padding, border, and the width of the element.
html {
font-size: 62.5%;
} 		This simple rule, which seems so simple, is so important, because browsers have the default font-size value of 16px, and when we set it to 62.5% of that value (10px) allows us to have a more precise measurement when using the flexible units (em, rem), defining that 1em equals to 10px.
